# Guestlytics

Guestlytics is a simple and elegant guest feedback web app connected to Supabase.

## Features

- Guest feedback form
- Rating and comments collection
- Supabase database integration
- Recent feedback display
- Simple responsive design

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file from `.env.example`:

```bash
cp .env.example .env
```

3. Add your Supabase keys:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

4. In Supabase, open SQL Editor and run:

```sql
create table feedback (
  id bigint generated by default as identity primary key,
  guest_name text not null,
  email text not null,
  rating int not null check (rating between 1 and 5),
  comments text not null,
  created_at timestamp with time zone default now()
);

alter table feedback enable row level security;

create policy "Anyone can submit feedback"
on feedback for insert
to anon
with check (true);

create policy "Anyone can read feedback"
on feedback for select
to anon
using (true);
```

5. Run the project:

```bash
npm run dev
```

## Deploy

You can deploy this project on Vercel. Add the same environment variables in Vercel project settings.
